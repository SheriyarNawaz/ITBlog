AI Chat bot

In this lesson we learnt how to create basic AI inside of python. At the start of the lesson our teacher showed us a variety of chat bots. We then were reminded how to control your program with the if statement. We created a program that askled a simple question to the user. The question was a yes no question to make it easier to make. We added input and a response. Finally we added a catch-all called else to say that if none of these are true i.e the answer is not yes or no, then just print something like "Invalid Input"

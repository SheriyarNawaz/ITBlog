Sorting Out Blog

In this lesson we spent some time sorting out and tidying up our blogs. I used to have a forked "Barry Clark" Jekyll repository. I then ran into an issue where I could not edit the HTML of the file. This was particularly a pain as I wanted to wrap my posts in a div to change the backround color and padding of the post. When I got home I decided to scrap the whole blog and start again but this time making a new repository and doing all the code by myself so I can have a lot more flexibility in the blog!
